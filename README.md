# ai-powered-trading-assistant
Production-ready LLM-powered chatbot for querying trading knowledge from PDFs using Hybrid Retrieval-Augmented Generation (RAG).

📌 Overview

This project is an AI-powered trading assistant that allows users to ask questions over large collections of PDFs and receive accurate, context-aware answers.

Unlike basic RAG systems, this solution uses:

Hybrid retrieval (FAISS + BM25)
Re-ranking (Cross-Encoder)
Context-grounded LLM responses

👉 Result: Better accuracy, reduced hallucination, and faster retrieval.
